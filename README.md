# ⌨️ 두유 노우 한메 타자교실?

**베드로의 네 장면을 손끝으로 새기는 한글 낙하 타자 대결 게임.**

`빌드 없음 / No build` · `Vanilla HTML/CSS/JS` · `WebAudio` · `localStorage 최고 기록`

</div>

---

옛날 한메타자교실 스타일로, 위에서 떨어지는 구절을 위험선에 닿기 전에 입력해서
지켜내는 타자 게임입니다. 베드로의 생애에서 네 장면 — 부르심, 고백, 부인,
회복 — 을 개역개정 원문 그대로 짧은 구절 단위로 잘라 스테이지로 구성했습니다.

## 🗺️ 네 장면

| # | 장면 | 본문 |
|---|------|------|
| 1 | 부르심 — 나를 따라오라 | 마태복음 4:18-22 |
| 2 | 고백 — 이 반석 위에 | 마태복음 16:13-18 |
| 3 | 부인 — 닭 울기 전에 | 누가복음 22:55-62 |
| 4 | 회복 — 내 양을 먹이라 | 요한복음 21:15-17 |

## 🎮 모드

- **1인 연습** — 네 장면을 차례로 클리어하며 점수·타/분(CPM)·완료율을 쌓고,
  최고 기록은 브라우저에 저장됩니다.
- **2인 대결** — 같은 순서로 두 사람이 번갈아 플레이(핫시트)하고, 스테이지별·
  총점 비교표로 승자를 가립니다.

난이도는 쉬움/보통/어려움 세 단계이며, 장면마다 낙하 속도가 다르게 조정되어
있습니다 (부인 장면이 가장 긴박하고, 회복 장면은 차분합니다).

## ▶️ 실행

빌드 과정이 없는 정적 페이지입니다. `index.html`을 바로 열거나 정적 서버로
띄우세요.

```bash
npx serve
```

PC 키보드 사용을 권장합니다 — 떨어지는 구절이 위험선에 닿기 전에 입력해서
지켜내세요.
