# ⌨️ 두유 노우 한메 타자교실?

**베드로의 네 장면을 손끝으로 새기는 한글 낙하 타자 게임.**

`빌드 없음 / No build` · `Vanilla HTML/CSS/JS` · `WebAudio` · `localStorage Top 5 리더보드`

</div>

---

옛날 한메타자교실 스타일로, 위에서 떨어지는 구절을 위험선에 닿기 전에 입력해서
지켜내는 타자 게임입니다. 베드로의 생애에서 네 장면 — 부르심, 고백, 부인,
회복 — 을 개역개정 원문 그대로 짧은 구절 단위로 잘라 스테이지로 구성했고,
네 장면을 모두 지나면 **보너스 스테이지 "이어지는 말씀"** 이 시작됩니다.

## 🗺️ 네 장면 + 보너스

| # | 장면 | 본문 |
|---|------|------|
| 1 | 부르심 — 나를 따라오라 | 마태복음 4:18-22 |
| 2 | 고백 — 이 반석 위에 | 마태복음 16:13-18 |
| 3 | 부인 — 닭 울기 전에 | 누가복음 22:55-62 |
| 4 | 회복 — 내 양을 먹이라 | 요한복음 21:15-17 |
| ★ | 보너스 — 이어지는 말씀 (무한) | 창세기부터 베드로전서까지 주요 구절 |

보너스 스테이지는 성경 곳곳의 주요 구절(개역개정)이 점점 빨라지는 파도로
끝없이 내려오는 서바이벌입니다. 각 구절 상자에 장절(예: 요 3:16)이 함께
표시되고, 하트 셋이 다할 때까지 몇 구절을 새기는지 겨룹니다.

## 🎮 플레이

- 시작 전에 **부서**(송림 청소년 / 서현 중등부 / 서현 고등부)와 **닉네임**
  (최대 4자)을 입력합니다.
- 떨어지는 구절을 입력한 뒤 **Enter** 또는 문장 끝의 **Space**로 확정해
  지켜냅니다. 위험선 가까이 내려온 구절은 붉게 깜빡입니다.
- 콤보를 이어가면 점수 보너스와 함께 칭찬이 터집니다.
- 난이도는 쉬움/보통/어려움 — 난이도가 높을수록 점수 배율이 붙고(×1 / ×1.15
  / ×1.35), 리더보드에 난이도가 함께 기록됩니다.
- 완주하면 총점에 따라 **칭호**를 받습니다: 그물 깁는 새싹부터
  ★ 반석 위의 타자왕까지.
- 최종 화면에 **역대 TOP 5**(닉네임·부서·난이도·점수)가 표시되고, 기록은
  이 브라우저에 저장됩니다.
- 게임 중 ⏸ 일시정지 / 🔊 소리 끄기, 일시정지에서 "처음으로"로 언제든
  다음 사람에게 넘겨줄 수 있습니다.

모바일 최적화: 세로 화면 권장, 가상 키보드 대응(자소 분리·조합 보정 포함).
